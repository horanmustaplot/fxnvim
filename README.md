### Installation

1. With ssh.
```bash
git clone --depth 1 git@github.com:horanmustaplot/fxnvim.git ~/.config/nvim 
```

1. With https.
```bash
git clone --depth 1 https://github.com/horanmustaplot/fxnvim.git ~/.config/nvim 
```
